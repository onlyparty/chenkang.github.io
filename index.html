php swoole扩展

#swoole 安装学习

安装包：?sudo git clone https://github.com/swoole/swoole-src.git

swoole 使用：可一步请求数据库 代码数据库使用可看swoole官网：
http://www.swoole.com/

编译生成configure文件

一 ： 通过phpize脚本生成
cd swoole-src.git
sudo /usr/bin/phpize7.0

如果没有找到phpize 先安装phpize

sudo apt-get install autoconf automake libtool m4
sudo apt-get install php7.0-dev
For PHP7.1, use php7.1-dev
二 ： 通过configure生成makefile 文件
sudo ./configure
三 ：通过 makefile 文件编译生成 .so文件

sudo make
sudo make test //可选
sudo make install
四 ：配置 .ini
cd /etc/php/7.0/mods-available/
touch swoole.ini
extension=swoole.so
sudo ln -s /etc/php/7.0/mods-available/swoole.ini/etc/php/7.0/cli/conf.d/swoole.ini
sudo ln -s /etc/php/7.0/mods-available/swoole.ini/etc/php/7.0/fpm/conf.d/swoole.ini
//fpm模式下需要重启fpm，phpinfo 查看
sudo service php7.0-fpm restart
php -m | grep swoole

git上传流程命令

##git常用命令

ssh-keygen -t rsa -C “youremail@example.com” 创建SSH Key
git config –global user.name “Your Name” 你名
git config –global user.email “email@example.com” 你邮箱（随意）
git clone ssh路径 连接远程（默认在桌面）
git流程
git status 查看状态
git add .文件名 准备上传的文件
git reset 文件名 撤销准备上传的文件
git commit -m ‘add 文件名’ 上传文件到本地仓库
git commit -am ‘edit b.php’
git log 查看日志
git push origin master:master 上传文件到远程仓库
git push origin 本地分支名：远程仓库分支名 把整个分支上传到远程分支
git pull 下载文件到本地仓库
git diff 查看冲突
